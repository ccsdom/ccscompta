Documentation du logiciel CCS Compta

Introduction

Bienvenue sur CCS Compta, votre plateforme de gestion comptable collaborative. Ce document vous guide à travers les fonctionnalités principales de l'application, que vous soyez un client, un comptable ou un administrateur.

Espace Client

Tableau de bord (Mes Documents)

C'est votre page d'accueil. Elle vous permet de téléverser de nouveaux documents et de suivre le statut de ceux que vous avez déjà envoyés.

  • Téléversement : Utilisez la zone de glisser-déposer pour ajouter de nouveaux fichiers (factures, reçus, etc.). Formats acceptés : PDF, JPG, PNG.
  • Liste des documents : Visualisez tous vos documents, classés par type (Factures, Reçus, etc.).
  • Statuts : Chaque document a un statut pour suivre son parcours :
    • En attente : Le document est téléversé mais pas encore analysé par l'IA.
    • En traitement : L'IA est en train d'analyser le document.
    • En examen : Le document a été traité et est en attente de validation par votre comptable.
    • Approuvé : Votre comptable a validé le document et ses données.
    • Erreur : Un problème est survenu lors du traitement.

Scanner un document

Utilisez l'appareil photo de votre téléphone ou de votre ordinateur pour numériser un document physique et l'envoyer directement sur la plateforme. C'est idéal pour les reçus et les factures papier lorsque vous êtes en déplacement.

Mon Analyse

Cette page vous donne un aperçu de vos finances, basé sur les documents approuvés par votre comptable. Vous pouvez y voir :

  • Le total de vos dépenses.
  • Vos principaux fournisseurs.
  • La répartition de vos dépenses par catégorie (fournitures, transport, etc.).

Commentaires sur un document

Pour communiquer avec votre comptable au sujet d'une pièce spécifique, cliquez sur le document dans la liste. Un panneau s'ouvrira, affichant une section "Commentaires" où vous pouvez poser des questions ou apporter des précisions.

Espace Comptable

Tableau de bord

Votre tableau de bord central vous donne une vue d'ensemble de l'activité de tous vos clients : nombre de documents en attente, validations récentes, activité par client, etc.

Gestion des clients

Créez de nouveaux dossiers clients, modifiez les informations des clients existants, ou importez une liste de clients via un fichier CSV.

Documents du client

Après avoir sélectionné un client, cette page vous montre tous ses documents. C'est ici que vous validez le travail de l'IA. Vous pouvez :

  • Vérifier les données : Comparez l'aperçu du document avec les données extraites par l'IA.
  • Corriger si besoin : Tous les champs sont modifiables.
  • Approuver : Une fois les données validées, approuvez le document pour l'intégrer à la comptabilité.
  • Envoyer à Cegid : Après approbation, vous pouvez envoyer les écritures vers votre logiciel de production.

Paramètres d'automatisation

Dans les paramètres, vous pouvez activer l'auto-approbation. Si l'IA est suffisamment certaine de la qualité de son extraction (vous pouvez définir le seuil de confiance), elle peut approuver automatiquement les documents, vous faisant gagner un temps précieux.
